练习：合成数据¶

你需要随机取随机张图片，然后将它们拼接成新的图片。

你需要设置20%的数据作为验证集，以保证模型没有过拟合。

import numpy as np
from sklearn.model_selection import train_test_split

n_class, n_len, width, height = 11, 5, 28, 28#why 11?

def generate_dataset(X, y):
    X_len = X.shape[0]
    
    X_gen = np.zeros((X_len, height, width*n_len, 1), dtype=np.uint8)
    y_gen = [np.zeros((X_len, n_class), dtype=np.uint8)for i in range(n_len)]#for i in range(n_len)#5,48000,11 #list of numpy array
    # TODO: 随机取1~5个数字，并拼接成新的图片
    for n in range(X_len):
        np.random.seed(n)
        random_num = np.random.choice(n_len-1, 1)
        choice = np.random.choice(X_len,random_num+1)
        for i in range(5):
            if i <= random_num:
                this_picture = X[choice[i]]
                label = y[choice[i]]
                X_gen[n,:,i*width:(i+1)*width,0] = this_picture
                y_gen[i][n][label] = 1
            else:
                y_gen[i][n][10] = 1
    return X_gen, y_gen

X_raw_train, X_raw_valid, y_raw_train, y_raw_valid = train_test_split(X_raw, y_raw, test_size = 0.2,random_state = 0)

X_train, y_train = generate_dataset(X_raw_train, y_raw_train)
X_valid, y_valid = generate_dataset(X_raw_valid, y_raw_valid)
X_test, y_test = generate_dataset(X_raw_test, y_raw_test)

问题 2¶

你为解决这个问题采取了什么技术？请详细介绍你使用的技术。

回答：
1.深度神经网络

  • 激活函数使用tanh效果较好
  • 优化选择rmsprop效果较好。对rmsprop只作了简单了解
  • 层之间利用了dropout，对模型结果有一定提升。dropout是一种正则化方法，在当前层按照给定的比率断开神经元的连接，从而达到正则化的效果
  • 值得一提的是，最初实验时没有对数据进行预处理。预处理之后模型结果有很大提升。

2.卷积神经网络
dnn效果一般，于是尝试cnn。

  • 激活函数、优化、正则化、预处理与上面基本相同。
  • 卷积核和池子大小参考网上的经验设定。
  • cnn迭代耗时太久，但可以看到2次迭代效果已远好于dnn

def acc(X,y,model):
    y_pred = model.predict(X)
    acc_all = []
    for j in range(n_len):
        acc = 0
        for i in range(y[0].shape[0]):
            predict_value = np.argmax(y_pred[j][i])
            real_value = np.argmax(y[j][i])
            if predict_value == real_value:
                acc += 1
        acc = float(acc) / y[0].shape[0]
        acc = round(acc,4)
        acc_all.append(acc)
    return acc_all

我们刚才得到了模型每个数字的准确率，现在让我们来计算整体准确率，按照完全预测正确数字序列的标准来计算。

        if acc_this == n_len:
